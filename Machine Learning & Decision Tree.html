Artificial Intelligence

Machine Learning & Decision Tree

During the lectures on Machine Learning, I have learned about the different types of Machine Learning such as Supervised, Unsupervised, and Reinforcement Learning. We have also delved into Decision Trees, which are a type of Supervised Learning algorithm used for classification problems. I learned that Decision Trees are constructed by recursively splitting the data into subsets based on the features that provide the most information gain. We also discussed how to evaluate the performance of Decision Trees using metrics such as accuracy, precision, and recall. Overall, I gained a deeper understanding of how Machine Learning algorithms work and how they can be applied to solve real-world problems.
