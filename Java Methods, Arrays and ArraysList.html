Java Methods, Arrays and ArraysList

During my lectures on Java programming, I learned about creating methods in Java, which are used to perform specific tasks in a program. I was able to create and implement various methods, such as 'getTotal' and 'isEven', to solve different problems. Additionally, I learned about the concept of arrays and how to declare, initialize, and manipulate them in Java. I also learned about the ArrayList class, which provides more flexibility in working with arrays by allowing the addition and removal of elements at runtime. Through practice exercises, I was able to solidify my understanding of these concepts and their practical applications in Java programming.
