HTML and CSS Structure
This is a reflective journal from the lecture which I have attended on the 27th of November 2021.

During my lectures, I learned about the structure of HTML and CSS. HTML provides the structure and content of a webpage, while CSS is responsible for the presentation and styling of the webpage. I learned about the various HTML tags, such as the header, footer, nav, and section tags, and how they can be used to create the structure of a webpage. Additionally, I learned about CSS selectors, properties, and values, and how they can be used to style HTML elements. I also learned about the box model and how it is used to determine the layout of a webpage. Understanding the structure of HTML and CSS is crucial in developing and designing a functional and aesthetically pleasing webpage.
