Elementary Java, Reading User Input and If Else Statements

Elementary Java is a foundational course in computer science that introduces students to basic programming concepts using the Java programming language. Throughout this course, I learned how to read user input and use if-else statements to create conditional logic in my programs. Reading user input is an important concept because it allows programs to interact with users in a meaningful way. If-else statements are a powerful tool for controlling the flow of a program based on certain conditions. By the end of the course, I gained a solid understanding of these fundamental programming concepts and how to apply them in Java. This knowledge provided me with a strong foundation for more advanced programming courses and projects.
